Friends of Princeton Open Space Riparian Restoration

A pilot project using open-source GIS software to facilitate two separate but related surveys conducted by Friends of Princeton Open Space (FOPOS) as part of the ongoing Riparian Restoration Project. Also generally intended to emphasize the value of GIS (esp. open source) for environmental stewardship efforts, relative to:

  • transect layout
  • navigation in the field
  • data collection
  • data analysis
  • data presentation
  • fun

Details

  • RES: original project proposal for the Rutgers Environmental Stewards program
  • Riparian East Botanical Survey
  • Riparian East Forest Inventory -- including carbon benefits generated by iTree

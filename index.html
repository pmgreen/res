Rutgers Environmental Stewards (RES)

Friends of Princeton Open Space Riparian Restoration Project

This is a pilot project using open-source GIS software to facilitate two separate but related surveys conducted by Friends of Princeton Open Space as part of the ongoing Riparian Restoration Project. It is intended also to emphasize the value of GIS for environmental stewardship efforts.

Click the links below for details and documentation.

  • Riparian East Botanical Survey
  • Riparian East Forest Inventory

This project was conducted as part of a [Rutgers Environmental Stewards](https://envirostewards.rutgers.edu/) certification project.

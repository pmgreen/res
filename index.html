Friends of Princeton Open Space Riparian Restoration

This is the overview page for a project focussed on using open-source GIS software (QGIS & QField) to facilitate two separate but related surveys conducted by Friends of Princeton Open Space (FOPOS) as part of the ongoing Riparian Restoration Project.

The general purpose was to emphasize the value of GIS for stewardship efforts, relative to:

  • laying out transects
  • navigation in the field
  • data collection
  • data analysis
  • data presentation

The Details

Project Proposal

  • Riparian East Botanical Survey: code and documentation for a data collection form for botanical surveys
  • Riparian East Forest Inventory: code and documentation for a data collection form for forest inventories; includes carbon benefits generated by iTree

This project was conducted as part of a Rutgers Environmental Stewards certification project.
